OllamaChat Support

Private AI chat for iOS — Connect to your own Ollama server or Ollama Cloud

Getting Started

  • Cloud mode: Go to Settings → enter your Ollama Cloud API key → select "Cloud" → choose a model and start chatting.
  • Local mode: Go to Settings → enter your Ollama server URL (e.g., 192.168.1.100 or a Cloudflare Tunnel URL) → select "Local" → choose a model.
  • Cloudflare Tunnel: To access your Ollama server remotely, you can use Cloudflare Tunnel to create a secure URL.

Requirements

  • iOS 17.0 or later
  • An Ollama server (local or remote) or an Ollama Cloud API key

Features

  • New Cloud & Local connection — Switch between cloud and local servers
  • AI Multi-model support — Switch between models with one tap

FAQ

How do I connect to my local Ollama server?
Go to Settings → enter your server's IP address (e.g., 192.168.1.100). The app automatically adds http:// and port 11434. For remote access, use a Cloudflare Tunnel URL.
What is a Cloud API key?
An Ollama Cloud API key lets you use cloud-hosted AI models without running your own server. You can get one at ollama.com.
Is my data collected by the app?
No. All chat data is stored locally on your device. The app developer has no access to your conversations, API keys, or server configurations. See our Privacy Policy for details.
How do I report a bug or request a feature?
Open an issue on our GitHub Issues page.
